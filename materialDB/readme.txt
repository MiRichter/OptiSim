MaterialDB folder for OptiSim
-----------------------------

Add your personal files here to see them in the DB list.
Format specifications:
	- choose unique file names, no dublicates
	- only .dat files are considered
	- 3-coloumn format: wavelength, n, k
	- write [mum] on top to change from nm to µm for wavelength

For further files see on www.refractiveindex.info database!!!
